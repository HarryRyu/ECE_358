1. Remove the extension from Makefile.txt (Make it into Makefile)
2. Convert all files starting with ece358_lab1_XXX from .txt to .py
3. Do "make setup" to install the necessary libraries
4. To run the question of choice, do `make run_qX`, where X is the question number. 
	1. Example: `make run_q1`

If running the program causes missing library errors, manually install the libraries.
	1. Install plotting package: `pip install matplotlib`
	2. Install statistics module: `pip install statistics`
